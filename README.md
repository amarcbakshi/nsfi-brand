# nSFi Brand & Design Assets

Official brand and design assets for **New Social Forms** (nSFi)—an initiative of the [MIT Center for Constructive Communication](https://www.ccc.mit.edu/) at the MIT Media Lab, made possible by Google Xi.

> New Social Forms brings together artists, technologists, researchers, and civic actors to explore, prototype, and scale the social architectures that strengthen connection and collective agency in a time of rapid technological and cultural change.
>
> → [newsocialforms.org](https://newsocialforms.org)

## Logos

All logo variants are available in SVG, PNG, and JPEG. SVG is preferred for web and print.

| File | Description |
|---|---|
| `Wordmark_1-line_black/white` | Full "New Social Forms" wordmark, single line |
| `Wordmark_Stacked_black/white` | Full wordmark, stacked layout |
| `Wordmark_Monogram_black/white` | Wordmark with monogram |
| `nSFi_MonogramBlack/White_rev` | Standalone nSFi monogram |
| `Avatar` | Square avatar / social profile image |

Use black variants on light backgrounds; white variants on dark backgrounds.

## Typography

### UN-11 (Primary)
UN-11ST by [SourceType](https://sourcetype.com/) is the primary typeface for all nSFi communications. Three weights are included: Regular, Italic, Bold — in OTF, TTF, WOFF, and WOFF2 formats.

**License:** UN-11ST is a commercially licensed font. See `Font/UN-11 Font/SourceType_EULA.pdf` for terms of use.

### TINY5x3 (Secondary / Accent)
TINY5x3 is used as an accent typeface for decorative and UI contexts. A variable TTF and static OTF instances at sizes 20–300 are included, along with web font formats (WOFF/WOFF2).
